## Mục tiêu

---

### Ngày 1 — Cấu trúc HTML

#### Học

* HTML là gì
* Browser đọc HTML như thế nào
* Cấu trúc cơ bản:

```html
<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>
```

#### Thực hành

Tạo:

```txt
index.html
```

Hiển thị:

* Tên
* Tuổi
* Sở thích

#### Bài tập

Tạo trang giới thiệu bản thân.
