Những gì học được trong site này

1. favicon: /images/favicon.ico là 1 url đúng, chỉ cần Ctrl + F5 ép page load lại là được.

2. trước giờ chỉ biết căn trái phải là css, bây giờ dùng HTML thì chỉ cần tạo 1 bảng, sau đó ẩn viền thì cũng có thể làm được.

3. đường dẫn tương đó phải bắt đầu bằng "/" nó được tính từ thư mục gốc, chúng ta có thể dùng luôn url tương đối, "/html css js/images/JuneHuhPicture2.jpg" và "./images/JuneHuhPicture2.jpg" thì đều được cả
